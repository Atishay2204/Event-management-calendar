Event management calendar

Overview
This project is a simple Python program designed to organize and manage personal schedules and events. It allows the user to add, view, reschedule, and cancel event records efficiently.

Features
-Add a new event (Date, Time, Description)
-View all scheduled events
-Update event details (Reschedule or edit location)
-Delete event information (Cancel an event)
-Stores data inside a list (In-memory database)

How to Run

2.Navigate to the folder where main.py is saved.
